# Bataille Navale - Projet BDW

Jeu de bataille navale avec cartes bonus/malus, développé dans le cadre de l'UE LIFBDW de deuxième année licence informatique à l'université de Claude Bernard Lyon 1.

## Description

Application web permettant de jouer à la bataille navale contre un adversaire IA. Le jeu intègre un système de cartes bonus/malus qui viennent enrichir le jeu classique.

## Structure du projet

```
BatailleNavale/
|-- controleurs/ # Contrôleurs 
|-- model/       # Accès base de données
|-- static/
|   |-- css/     # Feuilles de style
|   |-- img/     # Images 
|-- templates/   # Templates html
|-- init.py      # Initialisation de l'application
|-- routes.toml  # Définition des routes
```
